How to define a table in HTML

A table in HTML is defined by <table>

<tr> is used when adding a row in the table.

<th> is used as a header cell that is usually bolded and centered.

<td> is used for table data that includes the main content of each cell.

<thead> stands for "table head" and is used for grouping all the header rows together. It is usually used for styling.

<tfoot> is the "table footer" and can be used to group totals or summary info. It is found at the bottom of the table.

<caption> is used for adding a title or description at the top of the table.

What is the difference between colspan and rowspan?

colspan is used for merging columns to expand a cell horizontally. rowspan merges rows to expand a cell vertically.

What is the difference between cells spacing and padding?

Cell spacing is used to define the spacing between each table cell. Cell padding is to define the space within each cell.

How do you define table borders using CSS?

In CSS, you can define table borders by using the border element that determines the width in px, the style, and the color of the borders. The border-collapse property is used for combining cell borders into a single line.

How do you control border widths using CSS?

By using the "border-width" property in CSS, you can make border widths thicker ot thinner in px.
